# Hospital Appointment System

The Hospital Appointment System is a web application designed to facilitate the booking of medical appointments for patients by receptionists. It allows receptionists to schedule appointments with available doctors and send confirmation emails to patients. Additionally, it notifies patients if a doctor becomes unavailable.

## Features

- User-friendly web interface for receptionists to schedule appointments.
- Real-time doctor availability checks to prevent double-bookings.
- Automatic confirmation emails sent to patients upon successful appointment booking.
- Email notifications sent to patients in case of doctor unavailability.
- Secure storage of patient information and appointment records.

## Getting Started

### Prerequisites

- Python 3.x
- Django Web Framework
- Database system (e.g., PostgreSQL, MySQL, SQLite)
